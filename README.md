# Chess Game (C++)

A chess game implemented in **C++** with a graphical interface using **SFML**. The project was developed as part of a university C++ course and implements the main chess rules, including piece movement, check, checkmate, and stalemate detection.

## Build and Run

The project uses **CMake** as the build system.

### Build

```
mkdir build
cd build
cmake ..
cmake --build .
```

After building, the executable projectChess will be created. Make sure that the required textures and font files are available in the working directory, then run:

```
./projectChess
```
